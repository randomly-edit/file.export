How to Use the File Manager

This is a simple guide to help you get started.

  • Create File/Folder: Click the buttons in the toolbar to create new files or folders.
  • Context Menu: Press and hold on a file or folder for 1 second to open the context menu.
  • Rename/Delete: Use the context menu to rename or delete items.
  • Import/Export: Use the toolbar buttons to save your file structure to a file or load it from a file.
